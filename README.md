# Interactive To-Do List

## Project Description

This project is an interactive To-Do List built using HTML, CSS, and JavaScript. It demonstrates DOM manipulation by allowing users to add, complete, delete, and filter tasks. The application also displays a live counter showing the number of remaining tasks.

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

---

## Features

- Add new tasks
- Mark tasks as complete or undo them
- Delete tasks
- Live remaining task counter
- Filter tasks (All, Active, Completed)
- Responsive design
- DOM Manipulation using JavaScript

---

## JavaScript Concepts Used

- document.querySelector()
- addEventListener()
- createElement()
- appendChild()
- Arrays
- Objects
- Functions
- Event Handling
