# Animated Startup Landing Page

## Project Description

This project is a modern startup landing page created using HTML, CSS, and JavaScript. It demonstrates the use of CSS transitions, keyframe animations, and the Intersection Observer API to create smooth, engaging animations while maintaining a clean and responsive design.

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

---

## Features

- Responsive Navigation Bar
- Hero Section
- Features Section
- Call-to-Action Section
- Smooth Hover Transitions
- CSS Keyframe Animations
- Scroll Reveal Animation
- Responsive Design

---

## JavaScript Concepts Used

- Intersection Observer API
- querySelector()
- querySelectorAll()
- addEventListener()
- Class Manipulation

---

## CSS Concepts Used

- Flexbox
- CSS Grid
- Transitions
- Transform
- @keyframes
- Media Queries
